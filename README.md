# uts_rifky
CAP = Consistency – Availability – Partition tolerance
Teorema ini menyatakan bahwa dalam sistem terdistribusi, kita hanya bisa maksimal memilih 2 dari 3 sifat berikut sekaligus.
	•	Consistency (C): semua node memiliki data yang sama pada saat yang sama.
	•	Availability (A): sistem tetap responsif walau sebagian node gagal.
	•	Partition Tolerance (P): sistem tetap berjalan walau terjadi gangguan jaringan antar node.
